Fuzzify Pashto

A JavaScript library that creates regular expressions (regex) for fuzzy searching Pashto text (approximate string matching in Pashto)

Live Demo

The Problem

It can be difficult to search for words in Pashto texts or dictionaries because of variants or difficulties in spelling. This is because:

  • Certain sounds in Pashto can be written with many different letters. This is true for Arabic and Farsi loan words. For instance:
    • The 'z' sound can be spelled with any one of the following: ز ذ ض ظ.
    • The 's' sound can be spelled with any one of the following: س ص ث.
  • Certain sounds in Pashto can be difficult for non-native speakers to distinguish.
    • It may be hard for learners to hear the difference between the following pairs such as: ر/ړ, څ/س, and ځ/ز.
    • In addition to this, some dialects differences in the pronunciation of letters are lost completely. This can be true of ښ/خ, ځ/ز, and څ/س.
  • Spelling often changes based on dialect, area, and level of education. For example:
    • Some people may write "ګرځېدل" while others may write "گرزيدل".
    • While the proper dictionary of "how" is "څنګه", a learner may encounter the same word written as "سنگہ".

Because of all these reasons, it can be difficult to search for words based on sound, or a particular non-standard spelling.

The Solution

Search strings can be converted to regular expressions (regex) that can be used for fuzzy searching (approximate string matching) so that, for example:

A search for Will Match
گرزيدل ګرځېدل
سنگہ څنګه
انطزار انتظار
د پاره دپاره
مالوم معلوم
زبا ژبه
سڑے سړی

and vice versa.
